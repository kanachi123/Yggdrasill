# FenwickTree

Дерево Фенвика (двоичное индексированное дерево, англ. Fenwick tree, binary indexed tree, BIT) — структура данных, позволяющая быстро изменять значения в массиве и находить некоторые функции от элементов массива. Впервые описано Рябко Б.Я. в 1989 году(вкпедия 
https://ru.wikipedia.org/wiki/%D0%94%D0%B5%D1%80%D0%B5%D0%B2%D0%BE_%D0%A4%D0%B5%D0%BD%D0%B2%D0%B8%D0%BA%D0%B0)  

1. Fenwick Tree (Binary Indexed Tree)
Это структура данных, реализованная через массив.
Используется для префиксных сумм и обновлений.
“Дерево” — это логическая структура, а не реально связанные узлы.
Часто называют алгоритмом Фенвика, но правильнее — это структура с алгоритмом работы над ней.

Пример использования: HP персонажей, суммарный урон по диапазону, частые обновления массива.

2. Алгоритм Фенвика (Fenwick’s algorithm)
Иногда под этим имеют в виду конкретный способ работы с массивом, который позволяет строить BIT и делать суммирование/обновления.
То есть алгоритм — это инструкции, а дерево — это структура данных, где эти инструкции применяются.

```python

prices = [100, 200, 150, 300, 250, 400]

```
чтобы запросить сумму цен товаров на отрезке от товара с индексом a до товара с индексом b, мы можем использовать метод range_query.

```python

# Запрос суммы цен товаров на отрезке от 2-го товара до 5-го
total_price = fenwick.range_query(2, 5)
print("Общая сумма от 2 до 5:", total_price)  # 200 + 150 + 300 + 250 = 900

```
