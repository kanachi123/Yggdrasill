# ***Data Structures***


#### **Структура данных** — это способ организации и упорядочивания данных, который позволяет эффективно выполнять операции, такие как добавление, удаление, поиск и обновление элементов. Разные структуры данных подходят для разных задач, и их эффективность зависит от типа структуры и сложности операций

**Структуры данных можно разделить на подгруппы по видам**,я и так это сделал,но определим по своему

## Структуры данных бывают:
  1. линейные
  2. нелинейные
  3. ассоциативные
  4. многомерные
  5. специализированные
