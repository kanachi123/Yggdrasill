# ***DisjointSet(union–find)*** 

## **Определение: Пусть S конечное множество, разбитое на непересекающиеся подмножества(короче кластер из всех групп так в котором кто либо не является общим то есть множество так разбили на подмножества что общих элементов нет)**
### **S = $X_0$ $\cap$ $X_1$ $\cap$ $X_2$ $\cap$ ... $\cap$ $X_k$ : $X_i$ $\cup$ $X_j$ =  $\varnothing$ ,где для любых неотрицательных целых i,j i $\neq$ j**
### **для каждого подмножества $X_i$ назначается представитель $r_i$ $\in$ $X_i$**
1. makeset(x) - для x создает новое подмножеств в котором $r_i$ уже сам этот элемент
2. union(r,s) - return $E_k$ = $S_i \cap R_j$,где $r_i$ становится представителем нового множества
3. find(x) - определяет для подмножества x $\in$ S подмножество,к которому принадлежит этот элемент и return $r_i$

#### *представитель это как староста в факультете, он характеризует множество в контексте*
#### *например A = {1,2,3,4,5} $r_4$ = 5 ее можно выбрать представителем множества например множество в котором у тебя высший бал,ты там самый гигачад,ты станешь в гигачад контексте представителем(каждому можно дать ранк и потом сортировать по ней,как весы в графах)*
### ***очередная структура данных,которая используется в графах,короче ищет в множестве пересечения и обьеденяет группы,полезно когда нужно понять сущность относится ли к группе или нет***

### ***пример из жизни:ты строишь дерево связей,чтобы шиперить двух людей(типо они же классная пара сейчас устроим им так скажем общее местечно общую среду когорту)например генетик увидел тебя и знает одну дефффку с которым хочет шиперить потому что у вас гигачад генетическая совместимость но просто так сказать как то не красиво и ищет общую принадлежность к общей группе(множеству людей)***

### ***используется в Kruskal`s algorithms,но это потом здесь только структура данных а не алгоритмы.Еще например в компонентах связи или кластеризации но мы нубы пока в этом по этому не будет подробно кому интересно может сам найти(кластерныый анализ)***

```cpp
int Find(int x) 
{
    return ( x == p[x] ? x : p[x] = Find(p[x]) );
}

/* рекрусивно ищем представителя данного множества root короче в котором наше значение*/
/* если является уже представителем возвращаем значение в ином случаем через его представитель ищем его Find(p[x]),потом снова Find(p[p[x]])...пока не найдем,чтобы найти представителя этого корня и представитедю присваиваем значение поиска */

```

```cpp
void Union(int x, int y) 
{
    if ( (x = Find(x)) == (y = Find(y)) )
        return;//ищем представителей и сравниваем если совпадают значит общий рут
	
    if ( rank[x] <  rank[y] ) /* сравниваем у кого ранг выше тот для другого представитель в новом множестве */
        p[x] = y;
    else {
        p[y] = x;
        if ( rank[x] == rank[y] )//если одно и то же одно увеличиваем чтобы он стал рутом не важно какое важно чтобы кто то был ниже
            ++rank[x];
    }
}

```
